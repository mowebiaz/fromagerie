# Fromagerie (Static Next.js Showcase)

Static-exported showcase website for a mountain cheese shop (Haute-Savoie / ski resort).

## Pages
- `/` · `/produits` · `/raclettes` · `/fondues` · `/plateaux` · `/contact`

## Stack
Next.js 15, React 19, SCSS, Motion, React Aria Components, Leaflet, react-icons.

## Getting Started
```bash
npm install
npm run dev
```

## Build (static)
```bash
npm run build
```
